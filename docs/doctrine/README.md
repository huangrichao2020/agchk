# agchk Doctrine

This directory contains the long-horizon doctrine behind `agchk`.

`agchk` should stay usable as a tool, but its deeper value is methodological:

- a common language for agent architecture failures
- a reusable self-audit discipline
- a structured way for real-world agent problems to flow back into open source
